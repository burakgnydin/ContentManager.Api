# Content Manager API

A **backend content management API** developed with **C# and .NET** for managing articles, posts and digital content.

## Features

* Create and manage content
* Category management
* Content editing and deletion
* Database integration
* RESTful API design

## Technologies

* C#
* .NET
* REST API
* SQL Server
* Entity Framework

## How to Run

1 Clone the repository

```
git clone https://github.com/burakgnydin/content-manager
```

2 Open with **Visual Studio**

3 Restore NuGet packages

4 Run the API

## Possible API Endpoints

```
GET /contents
POST /contents

GET /categories
POST /categories

PUT /contents/{id}
DELETE /contents/{id}
```
